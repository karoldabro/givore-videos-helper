# Phrase Variations

## Core Principle
NEVER use the same phrasing twice in consecutive videos.

## App Demo Variations

### Speed-Focused
- "Veinte segundos y listo."
- "En el tiempo que tardas en hacer una foto de Instagram."
- "Más rápido que pedir un café."

### Ease-Focused
- "Más fácil que subir una story."
- "Foto, título, publicar. Ya."
- "Sin complicaciones. Sin registros largos."
- "La ubicación se pone sola."

### Result-Focused
- "Y ahora cualquiera cerca puede verlo."
- "Ya está visible para quien lo necesite."
- "En unos minutos, alguien del barrio lo puede recoger."

### Process Demos (When Showing App)
- "Mira, abro... pongo el título... foto... y publico."
- "Así de simple. Título, foto, listo."
- "Abro, subo, publico. Ya."

## App Introduction Variations

| Variation | Best For |
|-----------|----------|
| "Por eso yo uso Givore." | Standard (limit use) |
| "Y aquí es donde entra Givore." | Natural transition |
| "Pues para eso existe Givore." | Solution framing |
| "Lo bueno es que hay una manera de cambiar esto." | Hope focus |
| "¿Conocéis Givore?" | New audience |
| "Lo subo a Givore para que alguien lo aproveche." | Giving focus |

## Value Proposition Variations

- "Reciclaje social." (limit use)
- "Dar y encontrar."
- "Segunda vida para todo."
- "Vecinos conectando sin conocerse."
- "Conectar a quien tiene con quien necesita."
- "Lo que para unos sobra, para otros es justo lo que buscan."

## Emotional Triggers

### Discovery Excitement
- "¡Qué pasada!" / "¡Brutal!" / "¡Mira esto!" / "¡No me lo creo!"

### Empathy
- "Es una pena" / "Qué oportunidad perdida" / "Da penita"

### Satisfaction
- "Qué gusto" / "Satisfacción total" / "Mejor imposible"

### Community Pride
- "Vecinos conectando" / "Ahí está la magia" / "Entre todos"

## Words to AVOID

- **Formal**: "Sin embargo", "Por consiguiente", "Cabe destacar"
- **Marketing speak**: "Descarga ahora", "La mejor app", "Increíble oferta"
- **Generic weak**: "Interesante", "Bonito", "Bueno"
- **Accusatory**: "Nadie hace nada", "Es absurdo", "La gente no piensa"

## Mandatory Rotation

These must be rotated (NOT used in consecutive videos):
- App intro phrases
- App demo variations
- Value proposition phrases
- "Flipante" / "Increíble" alternatives
- Closing mood phrases
