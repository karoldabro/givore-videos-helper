# Tone Guardrails

## Core Principle: Viewer as ALLY

The viewer is your ALLY in solving the waste problem, NOT part of the problem.

Every script should make viewer feel:
- Joining a movement, not being lectured
- Discovering something cool, not being guilt-tripped
- Part of the solution, not the cause
- Inspired, not accused

## NEVER Use These Phrases (Accusatory)

| Phrase | Why Problematic |
|--------|-----------------|
| "Y nadie hace nada" | Direct blame on viewer/society |
| "La gente tira..." (generic) | Lumps viewer with "bad" people |
| "Es absurdo" | Implies people are stupid |
| "¿Por qué la gente...?" | Accusatory framing |
| "Nadie se da cuenta" | Viewer feels attacked |
| "La gente no piensa" | Condescending |

## USE These Instead (System-Focused)

| Instead of... | Use... |
|---------------|--------|
| "La gente tira cosas buenas" | "Se pierden cosas increíbles" |
| "Nadie hace nada" | "Falta una conexión" / "Ahí se quedan... esperando" |
| "Es absurdo" | "Es una pena" / "Qué oportunidad perdida" |
| "¿Por qué la gente...?" | "¿Sabéis qué pasa?" / "La cosa es que..." |
| "Nadie lo aprovecha" | "Solo falta que alguien lo encuentre" |

## Section-Specific Tone

### PROBLEM Section (8-20s)
- 70% empathy ("Te da pena", "Es una pena")
- 20% surprise ("Increíble lo que se encuentra")
- 10% gentle frustration (at SITUATION, not people)

### IMPORTANCE Section (20-25s)
- 80% opportunity ("Alguien lo aprovecharía")
- 20% community ("Entre todos")
- 0% guilt (NEVER)

**BAD**: "Hay familias ahí fuera que necesitan esto"
**GOOD**: "Alguien cerca estaría encantado de darle una segunda vida"

### RE-HOOK Section (25-28s)
- 100% empowerment ("Eso puede cambiar")
- 0% criticism

## Red Flag Words

If script contains these, rewrite that section:
- "absurdo" → judgmental
- "nadie" (negative context) → accusatory
- "la gente" + negative verb → blame
- "no entiendo cómo" → condescending
- "deberíais" → preachy
- "tenéis que" → commanding
