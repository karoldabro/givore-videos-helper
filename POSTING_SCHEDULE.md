# Posting Schedule Guide

Research-backed scheduling rules for the Givore channel across TikTok, Instagram Reels, and Facebook Reels.

**Sources**: Buffer 2026 study (2-5 posts/week optimal), TikTok Jan 2026 algorithm update (5+ same-format = suppression), internal batch pipeline constraints.

## Rules

1. Maximum 3-4 posts per week across the Givore channel
2. Never post the same format on consecutive days
3. Spread 7 batch variants across 10-14 days (not 2-3 days)
4. Maximum 1 CLASSIC_STREET_FINDS format per week
5. Mix narration styles: never post 3 full-narration videos in a row (alternate with minimal/zero/character)
6. Consistency over volume: same days each week is better than random bursts
7. Never post 5+ videos of the same format in a single week (triggers TikTok suppression)
8. Atmospheric/contemplative content (LLUVIA, SONIDOS, LO_QUE_NADIE_VE) works best on weekends

## Platform-Specific Guidelines

### TikTok

- **Best formats**: CUANTO_CUESTA, EL_RANKING_CALLEJERO, PREGUNTA_DEL_DIA, POV_ERES_MI_BICI, LO_MAS_RARO_DE_HOY
- **Algorithm priorities**: Watch time > views. Format diversity rewarded. 5+ same-format/week = suppression.
- **Post time**: 7-9 AM or 6-8 PM (Spain CET/CEST timezone)
- **Max frequency**: 3/week
- **Notes**: Short formats (under 30s) get near-100% completion rate boost. Price reveals and rankings drive highest comment velocity. Duet/stitch bait formats (PREGUNTA, BICI_VS_COCHE) extend reach through creator responses.

### Instagram Reels

- **Best formats**: LO_QUE_NADIE_VE, LLUVIA_NOCHE_EXTREMO, SONIDOS_DE_LA_CALLE, 60_SEGUNDOS_EN, ANTES_AQUI_HABIA
- **Algorithm priorities**: Saves and shares weigh heavily. Atmospheric/contemplative content outperforms.
- **Post time**: 6-8 AM or 7-9 PM (Spain CET/CEST timezone)
- **Max frequency**: 4/week (Reels)
- **Notes**: Longer narratives and atmospheric content perform better here than on TikTok. Educational content gets high save rates. Quiet flex aesthetic aligns with Instagram's 2026 direction.

### Facebook Reels

- **Best formats**: ANTES_AQUI_HABIA, EL_QUE_TIRA_GANA, 60_SEGUNDOS_EN, TRES_COSAS, EL_RANKING_CALLEJERO
- **Algorithm priorities**: Reels get 22% more organic reach than regular video. Community/local content performs well.
- **Post time**: 8-10 AM or 5-7 PM (Spain CET/CEST timezone)
- **Max frequency**: 4/week
- **Notes**: Older demographic (35-44 core) connects with nostalgia, community stories, and local guides. Cross-post from Instagram with adapted captions. Facebook comments tend toward longer discussions -- community formats thrive.

### Pinterest (Renueva channel only)

- **Best formats**: Before/after transformations, DIY ideas, home decor inspiration
- **Post time**: Anytime (pins have months-long lifespan, timing is less critical)
- **Max frequency**: 3-5 pins/week
- **Notes**: Not for the main Givore cycling channel. Renueva upcycling content only. Pins drive long-tail traffic for months. Optimize pin descriptions with keywords from KEYWORDS_RESEARCH.md.

## 7-Variant Batch Schedule Template

Spread 7 batch variants across 13 days minimum. Never post consecutive variants on consecutive days. Mix format types to send different algorithmic signals each post.

| Variant | Day | Format Category | Platform | Notes |
|---------|-----|-----------------|----------|-------|
| v1 | Day 1 (Mon) | CLASSIC_STREET_FINDS | IG, FB, TT | Flagship variant, full pipeline with all approval gates |
| v2 | Day 3 (Wed) | EL_RANKING / CUANTO_CUESTA | TT, IG, FB | High-viral format, different structure from v1 |
| v3 | Day 5 (Fri) | 60_SEGUNDOS / TRES_COSAS | TT, IG, FB | Educational, numbered structure |
| v4 | Day 7 (Sun) | LO_QUE_NADIE_VE / SONIDOS | IG, TT | Atmospheric, weekend mood content |
| v5 | Day 9 (Tue) | PREGUNTA / NO_TOQUEIS | TT, IG | Engagement-driven, comment bait |
| v6 | Day 11 (Thu) | LLUVIA_NOCHE / LO_MAS_RARO | IG, FB | Mood content, minimal narration |
| v7 | Day 13 (Sat) | POV_BICI / CUANTO_CUESTA | TT, IG | Experimental or high-viral closer |

### Batch Schedule Rules

1. v1 is always the CLASSIC format (uses full rotation system, updates global history)
2. v2-v7 should each use a DIFFERENT format from v1 and from each other when possible
3. At least 2 of v2-v7 should use minimal/zero/character narration (break the full-narration pattern)
4. At least 1 of v2-v7 should be a micro-format (under 30s) for completion rate diversity
5. Never schedule v1 and v2 on the same day or consecutive days
6. If a batch variant uses a series format (RANKING, 60_SEGUNDOS), check it does not clash with the regular series cadence
