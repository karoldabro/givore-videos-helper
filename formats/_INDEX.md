## File Naming Convention

Each format has its own file named after the format ID (e.g., `EL_RANKING_CALLEJERO.md`, `60_SEGUNDOS_EN.md`). Files contain all details: script sections, compatible structures/personas, clip guidance, platform priority, and algorithm signal.
