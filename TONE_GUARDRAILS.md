# Givore Tone Guardrails

Guidelines for maintaining positive, non-accusatory tone in viral scripts while still creating emotional engagement.

---

## Core Principle: Viewer as Ally

**The viewer is your ALLY in solving the waste problem, NOT part of the problem.**

Every script should make the viewer feel:
- They're joining a movement, not being lectured
- They're discovering something cool, not being guilt-tripped
- They're part of the solution, not the cause
- They're inspired, not accused

---

## Accusation Avoidance Rules

### NEVER Use These Phrases

| Accusatory Phrase | Why It's Problematic |
|-------------------|---------------------|
| "Y nadie hace nada" | Direct blame on viewer/society |
| "La gente tira..." (generic) | Lumps viewer with "bad" people |
| "Es absurdo" | Implies people are stupid |
| "¿Por qué la gente...?" | Accusatory framing |
| "Nadie se da cuenta" | Viewer feels attacked |
| "La gente no piensa" | Condescending |
| "Es increíble lo que hace la gente" | Judgmental |
| "¿Cómo es posible que...?" | Blame-seeking question |

### USE These Instead (System-Focused)

| Instead of... | Use... |
|---------------|--------|
| "La gente tira cosas buenas" | "Se pierden cosas increíbles" |
| "Nadie hace nada" | "Falta una conexión" / "Y ahí se quedan... esperando" |
| "Es absurdo" | "Es una pena" / "Qué oportunidad perdida" |
| "¿Por qué la gente...?" | "¿Sabéis qué pasa?" / "La cosa es que..." |
| "Nadie lo aprovecha" | "Solo falta que alguien lo encuentre" |
| "La gente no sabe" | "A veces no sabemos" (inclusive) |

## Emotion Calibration Guide

### Acceptable Emotions (USE)

| Emotion | How to Express | Example |
|---------|----------------|---------|
| **Surprise/Wonder** | Express YOUR reaction | "¡No me lo puedo creer!" |
| **Empathy** | Share the feeling | "Te da pena, ¿sabes?" |
| **Excitement** | Genuine enthusiasm | "¡Esto es una pasada!" |
| **Hope** | Focus on solution | "Y eso se puede cambiar" |
| **Community** | Inclusive language | "Esto lo creamos entre todos" |

### Problematic Emotions (AVOID or REFRAME)

| Emotion | Problem | Reframe To |
|---------|---------|------------|
| **Outrage at people** | Blames viewer | Outrage at SITUATION |
| **Disappointment in others** | Condescending | Sadness about lost opportunity |
| **Judgment** | Alienates audience | Curiosity about why |
| **Frustration** | Negative energy | Determination to change |

## Section-by-Section Tone Guidelines

### PROBLEM Section

**Goal:** Create emotional resonance WITHOUT blaming

**BAD (Accusatory):**
```
"Es que me parece absurdo. La gente tira cosas que funcionan perfectamente...
cuando hay familias ahí fuera que las necesitan. Y nadie hace nada."
```

**GOOD (System-Focused):**
```
"Es que me da pena. Cosas que funcionan perfectamente... acabando en la basura.
Y ahí se quedan, esperando a que alguien las encuentre."
```

**Key Differences:**
- "Me parece absurdo" → "Me da pena" (empathy vs. judgment)
- "La gente tira" → "Cosas acabando" (passive, no blame)
- "Nadie hace nada" → "Ahí se quedan, esperando" (situation, not people)

---

### IMPORTANCE Section

**Goal:** Show why this matters WITHOUT guilt-tripping

**BAD (Guilt-Inducing):**
```
"Hay familias ahí fuera que necesitan esto. Gente que no puede permitirse
comprar muebles. Y mientras tanto, vosotros pasáis de largo."
```

**GOOD (Opportunity-Focused):**
```
"Y mientras tanto, hay gente cerca que estaría encantada de darle una segunda vida.
Solo falta conectarlos."
```

**Reframing Patterns:**

| Guilt Frame | Opportunity Frame |
|-------------|-------------------|
| "Familias que necesitan" | "Alguien que lo aprovecharía" |
| "Gente que no puede comprar" | "Alguien buscando justo esto" |
| "Esto nos afecta a todos" | "Esto lo podemos cambiar" |
| "El planeta sufre" | "Podemos hacer algo bueno" |
| "Desperdicio innecesario" | "Segunda vida posible" |

---

### RE-HOOK Section

**Goal:** Recapture attention with EMPOWERMENT, not criticism

**BAD:**
```
"¿Y sabéis qué es lo peor? Que esto sigue pasando y nadie hace nada."
```

**GOOD:**
```
"Pero fijaos... eso se puede cambiar. Y os voy a enseñar cómo."
```

**Empowerment Phrases:**
- "Pues eso puede cambiar. Hoy. Con algo muy simple."
- "Pero aquí viene lo bueno."
- "Y esto tiene solución. Mirad."
- "Pues fijaos lo fácil que es hacer algo."

## Positive Language Substitution Bank

### For Problem Description

| Negative | Positive Alternative |
|----------|---------------------|
| "Tiramos" | "Se pierden" |
| "Desperdiciamos" | "Dejan de aprovecharse" |
| "Nadie recicla" | "Falta la conexión" |
| "La basura aumenta" | "Hay oportunidades sin aprovechar" |
| "El problema es gordo" | "Hay mucho potencial" |

### For Describing Waste

| Judgmental | Neutral/Empathetic |
|------------|-------------------|
| "Es un escándalo" | "Es una pena" |
| "Es absurdo" | "Qué oportunidad perdida" |
| "No tiene sentido" | "Podría tener otra vida" |
| "Es de locos" | "Increíble lo que se encuentra" |

### For Viewer Engagement

| Exclusive/Accusatory | Inclusive/Empowering |
|---------------------|---------------------|
| "Vosotros no sabéis" | "A lo mejor no sabíais" |
| "La gente no mira" | "Muchos pasamos sin fijarnos" |
| "Nadie se para" | "Es fácil pasar de largo" |
| "Tenéis que cambiar" | "Podemos cambiar esto juntos" |

## Emotional Balance Formula

Each script should have this emotional balance:

```
PROBLEM Section (8-20s):
- 70% empathy ("Te da pena", "Es una pena")
- 20% surprise ("Increíble lo que se encuentra")
- 10% gentle frustration (at situation, NOT people)

IMPORTANCE Section (20-25s):
- 80% opportunity ("Alguien lo aprovecharía")
- 20% community ("Entre todos")
- 0% guilt (NEVER)

RE-HOOK Section (25-28s):
- 100% empowerment ("Eso puede cambiar")
- 0% criticism

SOLUTION Section (28-45s):
- 60% excitement about app
- 40% ease/simplicity
- 0% pressure to download
```

---

## Quick Tone Check Before Output

Before generating final script, verify:

- [ ] No direct blame on "la gente" or viewers
- [ ] Problem framed as SITUATION, not PEOPLE
- [ ] Importance shows OPPORTUNITY, not GUILT
- [ ] Re-hook uses EMPOWERMENT language
- [ ] Overall feeling: inspiring, not lecturing
- [ ] Viewer feels like ALLY, not target

---

## Red Flag Words to Search For

If your script contains these words, rewrite that section:

| Red Flag | Likely Problem |
|----------|---------------|
| "absurdo" | Judgmental |
| "nadie" (in negative context) | Accusatory |
| "la gente" + negative verb | Blame |
| "no entiendo cómo" | Condescending |
| "deberíais" | Preachy |
| "tenéis que" | Commanding |
| "es increíble que" | Judgmental surprise |

---

## Input Field for Script Generation

When requesting a script, specify:
```
Tone: educational | exciting | community | emotional
```

- **educational:** Focus on teaching how things work
- **exciting:** Focus on discovery and surprise
- **community:** Focus on connection and movement
- **emotional:** Focus on empathy and hope (NOT guilt)
