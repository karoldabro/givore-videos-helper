# Givore Cross-Platform Strategy
## One Video, One Tool (Metricool), Five Platforms

---

## Your Workflow

```
1. Record video (GoPro cycling / phone)
2. Edit in Kdenlive (one version)
3. Generate voiceover (ElevenLabs)
4. Run script through Claude Project → Get all 5 platform descriptions
5. Upload to Metricool → Paste each platform's description
6. Schedule → Done
```

**Total extra work per video**: ~5 minutes (copy-paste descriptions in Metricool)

## Unified Keyword Strategy (Consistency Across Channels)

| Context | Use These Terms | NEVER Use |
|---------|----------------|-----------|
| Social media (all) | "segunda vida", "consumo consciente", "se regala" | "gratis", "free", "reciclaje social" (as hashtag) |
| Web SEO | "cosas gratis", "muebles segunda mano", "segunda mano [city]" | — (commercial terms OK for web) |
| App Store | "cosas gratis cerca", "donar objetos", "regalar cosas" | — |
| Brand identity | "reciclaje social", "dar y encontrar" | — (keep in descriptions, not hashtags) |

**Rule**: "gratis" drives web SEO (39K/mo) but is commercially deprioritized on social platforms. Use it on web, avoid on social.

## Geographic Expansion Strategy

Content presents NATIONAL concept, not city-specific stories.
Hyperlocal targeting via hashtags, not title/description lockdown.

### Hashtag City Rotation for Expansion
- Week 1: #Madrid (250K "segunda mano madrid")
- Week 2: #Barcelona (170K)
- Week 3: #Valencia (140K)
- Week 4: #Sevilla (110K)
- Rotate: Malaga (93K), Zaragoza (93K), Bilbao (12K), etc.

When video shows a SPECIFIC city: use that city in hashtags + mention naturally.
When video is GENERIC (app demo, concept): rotate target city hashtags.

## Performance Patterns (Quarterly Review)

### Hook Types That Work (IG + TikTok data, Jan-Mar 2026)
1. **OUTRAGE/SURPRISE**: Highest views (159K avg IG, 213K TikTok top)
2. **COMMUNITY**: Highest engagement rate (3.15%) and shares (250 avg)
3. **URGENCY**: "Ayer", "Hoy" — creates FOMO, strong engagement

### Hook Types to Reduce
- **MYSTERY** ("tesoros", "esconde"): Overused (13/60 posts), 2.8K avg views
- **ITEM-FOCUSED** without emotional angle: Most common (19/60) but mediocre (6.4K avg)
