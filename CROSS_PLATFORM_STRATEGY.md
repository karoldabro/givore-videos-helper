# Givore Cross-Platform Strategy
## One Video, One Tool (Metricool), Five Platforms

---

## Your Workflow

```
1. Record video (GoPro cycling)
2. Edit in Kdenlive (one version)
3. Generate voiceover (ElevenLabs)
4. Run script through Claude Project → Get all 5 platform descriptions
5. Upload to Metricool → Paste each platform's description
6. Schedule → Done
```

**Total extra work per video**: ~5 minutes (copy-paste descriptions in Metricool)

---

## What's the Same (All Platforms)

| Element | Spec |
|---------|------|
| Video file | 1080x1920, 30fps, MP4, 45-60s |
| Audio | Same voiceover + music |
| Edit | Same cuts, same pacing |
| Captions | Same .srt file (Metricool adds them) |

---

## What's Different (Per Platform)

| Platform | Title | Description | Hashtags |
|----------|-------|-------------|----------|
| TikTok | Emoji + curiosity | Casual, short | 5 |
| Instagram | Aesthetic | Lifestyle tone | 7-10 |
| YouTube | SEO keywords | Searchable | 5-7 |
| Facebook | Community feel | Longer OK | 5 |
| LinkedIn | Professional/stat | Business value | 5 |

---

## Metricool Setup

### In Metricool's Multi-Platform Post:
1. Upload ONE video
2. Toggle "Customize per network"
3. Paste platform-specific title/description from Claude output
4. Paste platform-specific hashtags
5. Set optimal times (or use Metricool's "Best Time")
6. Schedule

### Metricool Best Times Feature
Let Metricool auto-select times OR use these:
- TikTok: 7:00 AM Spain
- Instagram: 12:00 PM Spain
- YouTube: 8:00 PM Spain
- Facebook: 3:00 PM Spain
- LinkedIn: 8:00 AM Spain
