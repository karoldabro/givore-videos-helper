# Givore Phrase Variations

Banks of alternative phrases to prevent repetitive, commercial-sounding scripts.

---

## Core Principle

**NEVER use the same phrasing twice in consecutive videos.**

Each script should feel fresh and authentic, not like reading from a template.

---

## 1. App Demo Variations

These replace the formulaic "Abro Givore, pongo título, hago fotos... en menos de veinte segundos."

### Speed-Focused Demos
| Variation | Best For |
|-----------|----------|
| "Veinte segundos y listo." | Quick mention |
| "En el tiempo que tardas en hacer una foto de Instagram." | Relatable comparison |
| "Más rápido que pedir un café." | Casual tone |
| "Un minuto y ya está visible para todo el barrio." | Impact focus |

### Ease-Focused Demos
| Variation | Best For |
|-----------|----------|
| "Más fácil que subir una story." | Youth audience |
| "Foto, título, publicar. Ya." | Ultra minimal |
| "Sin complicaciones. Sin registros largos. Solo subes y listo." | Barrier removal |
| "La ubicación se pone sola. Ni eso tienes que hacer." | Automation highlight |

### Result-Focused Demos
| Variation | Best For |
|-----------|----------|
| "Y ahora cualquiera cerca puede verlo." | Reach emphasis |
| "Ya está visible para quien lo necesite." | Benefit focus |
| "Ahora alguien que lo busca lo puede encontrar." | Connection focus |
| "En unos minutos, alguien del barrio lo puede ver." | Local impact |

### Process Demos (When Showing App)
| Variation | Best For |
|-----------|----------|
| "Mira, abro... pongo el título... foto... y publico." | Screen recording |
| "Así de simple. Título, foto, listo." | Quick demo |
| "Fijaos. [Título]. [Foto]. [Publicar]. Hecho." | Step by step |
| "Abro, subo, publico. Ya." | Ultra fast |

### Casual Demos
| Variation | Best For |
|-----------|----------|
| "Lo subo a Givore y me olvido." | Effortless vibe |
| "Una foto rápida, lo publico, y alguien lo descubre." | Flow |
| "Lo dejo en Givore y ya vendrá quien lo necesite." | Trust in system |

## Variation Tracking

### How to Use This Document

1. **Before writing:** Check what phrases were used in last 2-3 videos
2. **During writing:** Select alternatives from banks, don't default to most common
3. **After writing:** Verify no phrase repetition with recent content

### Mandatory Rotation

These phrases must be rotated and NOT used in consecutive videos:

| Category | Rotate These |
|----------|--------------|
| App intro | "Por eso uso Givore" variations |
| App demo | Speed/ease/result variations |
| Value prop | "Reciclaje social" variations |
| Closing | "El día redondo" variations |
| Emotion | "Flipante" / "Increíble" alternatives |
