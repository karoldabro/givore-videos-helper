# Script Structures — Rotatable Narrative Formats

6 alternative script structures to prevent cross-session monotony. Each video MUST use a structure different from the last 2 scripts. Track via `script-add --structure-type`.

**Core rule:** The structure defines SECTION ORDER and PACING. Content rules (tone guardrails, giveaway-first framing, no trash encouragement) apply to ALL structures equally.

---

## Structure Selection Decision Tree

```
What type of content is this?
|
+-- Multiple items found          --> COUNTDOWN or CLASSIC
+-- Single impressive item        --> COLD OPEN or MICRO-NARRATIVE
+-- Community/sharing focus       --> LOOP or PSP
+-- Quick find, simple story      --> MICRO-NARRATIVE
+-- App demo is important         --> CLASSIC or PSP
+-- Exploration/journey video     --> CLASSIC or COLD OPEN
|
Check last 2 structures used:
+-- Same as either? --> Pick a different one
+-- All different?  --> Free choice based on content fit
```

---

## 1. CLASSIC

The original 8-section format. Use as ONE option among six, not the default.

**Section order:**
```
HOOK (0-3s) --> PROOF TEASE (3-8s) --> PROBLEM (8-20s) --> IMPORTANCE (20-25s)
--> RE-HOOK (25-28s) --> SOLUTION (28-45s) --> PAYOFF (45-55s) --> CTA (55-60s)
```

**Timing:** Flexible within ranges. Not percentage-locked.

**Emotional arc:** Curiosity --> Empathy --> Empowerment --> Satisfaction --> Action

**Best for:** Exploration videos, neighborhood rides, when you want the full narrative build.

**When NOT to use:** Short finds, items that speak for themselves, when you've used CLASSIC in the last 2 scripts.

### Example (60s, ~200 words)

```
Tres cosas. He encontrado tres cosas... y cuando veais en que estado, vais a flipar.

Quedaos hasta el final porque hay algo que puede cambiar como veis vuestra ciudad.

Es que me da pena. Cosas perfectas que acaban ahi, esperando a que alguien las encuentre. Muebles que podrian durar anios... y nadie sabe que estan.

Y mientras tanto, hay gente cerca que estaria encantada de darles una segunda vida. Solo falta conectarlos.

Pero eso lo podemos cambiar. Mirad.

Dos sillas de madera. Una mesita. Perfectas las tres. Y lo que hago yo es subirlas a Givore. Foto, titulo, publico. Ya esta visible para todo el barrio.

Y ahora cualquiera cerca puede verlas y darles una segunda vida. Vecinos conectando sin conocerse.

Y bueno... mirad donde he acabado. El Turia. Que gusto.

Y vosotros? Que habeis encontrado en la calle ultimamente? Contadmelo en comentarios.
```

## 2. COLD OPEN

Start mid-action. No preamble, no hook buildup. The item IS the hook.

**Section order:**
```
ITEM REVEAL (0-5s) --> CONTEXT (5-12s) --> PROBLEM embedded (12-20s)
--> SOLUTION woven in (20-35s) --> REFLECTION (35-50s) --> CTA (50-60s)
```

**Key difference:** No separate PROOF TEASE or RE-HOOK. Problem and solution interweave naturally instead of being isolated sections.

**Timing:** Front-loaded. The first 5 seconds show the item. Builds context after the visual hook.

**Emotional arc:** Surprise --> Context --> Understanding --> Connection --> Action

**Best for:** Single impressive finds, items that are visually striking, proof-first videos. Stronger for experienced audiences fatigued by "Sabeis que?"

**When NOT to use:** Multi-item finds (too rushed), abstract topics, when item isn't visually interesting.

### Example (50s, ~165 words)

```
Esto estaba aqui hace cinco minutos. Un sofa. En la acera. Perfecto.

Estoy en la Zaidia, volviendo en bici del trabajo. Y ahi estaba. Alguien se ha mudado, supongo. O simplemente ya no lo queria.

Lo triste es que esto no tendria que estar aqui. Si hubieran sabido que alguien lo quiere... no habria acabado mojandose en la acera. Habria ido directo a su nueva casa.

Yo lo que hago es sacarlo en Givore. Una foto, dos palabras. Y alguien del barrio lo ve. Asi de simple. En un rato, este sofa puede estar en el salon de alguien que lo necesita.

Es que... cuando ves cosas asi, piensas: solo faltaba el puente. Solo faltaba que alguien conectara al que lo deja con el que lo quiere.

Antes de dejar algo en la calle, pensad si alguien lo aprovecharia. Seguro que si.
```

## 3. LOOP FORMAT

The ending connects to the beginning. The first line only makes full sense AFTER watching the whole video. Drives rewatches (algorithm boost).

**Section order:**
```
CRYPTIC OPENER (0-3s) --> DISCOVERY (3-15s) --> PROBLEM + CONTEXT (15-25s)
--> SOLUTION (25-40s) --> RETURN TO OPENER with new meaning (40-50s) --> CTA (50-60s)
```

**Key difference:** First and last lines mirror each other. Viewer rewatches to catch the connection.

**Timing:** The opener must be short and intriguing. The return moment is the emotional peak.

**Emotional arc:** Confusion --> Discovery --> Understanding --> Resolution --> "Aha!" --> Action

**Best for:** Items with a story, finds that changed perspective, community-focused content. When you want high rewatch rates.

**When NOT to use:** Straightforward finds, when the story is simple, items without narrative potential.

### Example (55s, ~180 words)

```
"Solo faltaban diez minutos."

Hoy iba en bici por Benimaclet y me encontre una mesa redonda. Sin un rasgunio. Perfecta. En la acera, al lado de un contenedor.

Alguien la dejo ahi. Y no voy a juzgar a nadie, porque a lo mejor no sabia que hacer con ella. Pero... alguien a dos calles de aqui la necesitaba. Y no lo sabia.

Entonces hice lo que hago siempre. La subi a Givore. Foto rapida, "mesa redonda, buen estado, Benimaclet." Publique.

Y sabeis que? A los diez minutos, alguien del barrio ya habia escrito. Diez minutos entre dejarlo en la acera y encontrar duenio nuevo.

Solo faltaban diez minutos. Eso es lo que separaba a esa mesa del vertedero y de una casa nueva. Diez minutos y una conexion.

Si teneis algo en casa que ya no usais... esos diez minutos pueden cambiar su historia. Subidlo.
```

## 4. MICRO-NARRATIVE

Single emotional arc. 30 seconds max. One item, one moment, one feeling. Minimal structure, maximum impact.

**Section order:**
```
MOMENT (0-8s) --> FEELING (8-15s) --> ACTION (15-22s) --> CLOSE (22-30s)
```

**Key difference:** No PROBLEM, no IMPORTANCE, no RE-HOOK. Strips everything to essence. The brevity IS the style.

**Timing:** 20-35 seconds total. Word count: 60-100 words max.

**Emotional arc:** Single wave. One emotion, rising and falling.

**Best for:** Quick finds, items that tell their own story, when you want punchy content, Instagram Reels (shorter format), testing new hooks.

**When NOT to use:** Complex stories, when you need to explain the app in detail, multi-item finds.

### Example (25s, ~75 words)

```
Un cesto de colada. En la acera. Perfectamente bien.

Alguien lo dejo aqui. Y alguien lo va a encontrar.

Lo subi a Givore. Diez segundos. Ya esta visible.

Es que no hace falta mas. No hace falta un discurso. Solo conectar. Alguien que lo deja, alguien que lo quiere. Un puente de diez segundos.

Asi de facil.
```

### Example 2 (30s, ~90 words)

```
Esto estaba aqui esta maniana. Ya no.

Una silla de madera en Russafa. La vi pedaleando. Le hice foto. La subi.

A la hora, alguien del barrio se la llevo. Una hora.

No se quienes son. No nos hemos visto la cara. Pero esa silla ahora esta en una casa en vez de en la basura.

Es lo unico que hace falta. Que la persona correcta lo vea en el momento correcto. Para eso esta Givore.
```

## 5. PSP (Problem-Solution-Proof)

Three-beat structure. Tight, argumentative, persuasive. No fluff, no scenic payoff, no re-hook.

**Section order:**
```
HOOK (0-3s) --> PROBLEM (3-18s) --> SOLUTION (18-35s) --> PROOF (35-50s) --> CTA (50-60s)
```

**Key difference:** PROOF replaces PAYOFF. Instead of scenic satisfaction, you show evidence that the solution works. More argumentative, less emotional.

**Timing:** Problem gets more time than in CLASSIC. Solution is followed by concrete proof.

**Emotional arc:** Attention --> Concern --> Hope --> Conviction --> Action

**Best for:** App demos with real results, when you have real Givore stories to share, persuasion-focused content, community-building videos.

**When NOT to use:** Scenic rides, when the journey matters more than the argument, purely emotional content.

### Example (55s, ~180 words)

```
En Espania se pierden cosas increibles cada semana. Cosas que funcionan.

Y es que... esto es lo que me encuentro cada vez que salgo en bici. Un somier en perfecto estado. Juguetes. Una mesa sin un rasgunio. En la acera. Todo el mundo pasa de largo.

Lo peor no es que esten ahi. Lo peor es que alguien que los necesita esta a quinientos metros y no lo sabe. Solo falta un puente.

Ese puente es Givore. Yo subo lo que encuentro: foto, titulo, barrio. En diez segundos esta visible para cualquiera cerca. Sin dinero. Sin complicaciones. Solo compartir.

Y funciona. La semana pasada subi una comoda en Torrefiel. A la media hora, una chica del barrio vino a buscarla. Media hora entre la acera y una casa nueva.

Hay usuarios en Givore que recogen muebles, los arreglan y los donan. Esto no es teoria. Esta pasando.

Antes de tirar algo, subidlo. Diez segundos pueden cambiar la historia de ese objeto.
```

## 6. COUNTDOWN / LIST

"N cosas que encontre hoy." Energy builds with each item. Works like a listicle — each item is a mini-reveal.

**Section order:**
```
HOOK + COUNT TEASE (0-5s) --> ITEM 1 (5-15s) --> ITEM 2 (15-25s) --> ITEM 3 (25-35s)
--> MINI-SOLUTION (35-45s) --> REFLECTION (45-52s) --> CTA (52-60s)
```

**Key difference:** No separate PROBLEM/IMPORTANCE — each item carries its own micro-narrative. SOLUTION is compressed to a "mini" version after all items are shown.

**Timing:** Each item gets equal time. Energy should INCREASE with each reveal.

**Emotional arc:** Anticipation --> Surprise x3 --> Understanding --> Action

**Best for:** Multi-item finds (3-5 items), exploration videos, neighborhood tours, "one hour on the bike" format.

**When NOT to use:** Single items, abstract topics, when the story matters more than the items.

### Example (55s, ~180 words)

```
Tres cosas. Una hora en bici. A ver.

Primero: Benimaclet. Un estante de madera al lado de un portal. Perfecto. Ni una marca. Alguien se ha mudado y ha dejado esto aqui. Si hubieran sabido que alguien lo quiere...

Segundo: cerca del Turia. Juguetes. Un monton. Funcionan todos. Un ninio lo pasaria en grande con estos. Pero estan aqui, en la acera, esperando.

Tercero: y esta es la buena. Russafa. Una mesa redonda. Sin un rasgunio. La toco... solida. Perfecta.

Tres cosas. Tres oportunidades perdidas. O no.

Porque yo las subo a Givore. Foto, titulo, barrio. Y en minutos la gente cerca puede verlas. Sin complicaciones. Solo conectar a quien tiene con quien necesita.

Tres cosas en una hora. Imaginad una ciudad entera compartiendo lo que ya no usa. Nada acabaria en la basura.

Que habeis encontrado vosotros esta semana? Contadme.
```

## Rotation Rules

1. **NEVER** use the same structure in 2 consecutive scripts
2. **Verify** via `givore-tools.sh script-rotation` before selecting
3. **Suggested rotation**: CLASSIC --> COLD OPEN --> LOOP --> MICRO --> PSP --> COUNTDOWN --> (repeat)
4. **Within a 7-variant batch**: v1 sets the structure. v2-v7 CAN use the same structure (they differentiate via other matrix dimensions). But if the batch includes MICRO variants, use at most 2 MICROs.

---

## Timing Guidelines (NOT Rules)

Each structure has suggested timing. Adapt to your content:

- If the item is visually stunning --> Front-load the reveal (COLD OPEN timing)
- If the story has a twist --> Save it (LOOP timing)
- If the find is simple --> Keep it short (MICRO timing)
- If you have proof/results --> Allocate time for evidence (PSP timing)

**NEVER force content to fit timing.** Let the story decide.

## Emotional Arc Variety

| Structure | Primary Emotions | Secondary |
|-----------|-----------------|-----------|
| CLASSIC | Curiosity, empathy, hope | Community pride |
| COLD OPEN | Surprise, understanding | Connection |
| LOOP | Confusion, discovery, "aha!" | Satisfaction |
| MICRO | One dominant emotion (varies) | None — keep pure |
| PSP | Concern, hope, conviction | Evidence-based trust |
| COUNTDOWN | Anticipation, surprise x3 | Cumulative impact |

**Rule:** If your last 3 scripts all used empathy+hope, pick a structure with a DIFFERENT emotional arc.

## Structure Compatibility with Content Pillars

| Content Pillar | Best Structures |
|---------------|----------------|
| Cycling POV / exploration | CLASSIC, COLD OPEN, COUNTDOWN |
| Single item find | COLD OPEN, MICRO, PSP |
| Multiple items | COUNTDOWN, CLASSIC |
| Community/sharing message | LOOP, PSP |
| Quick daily find | MICRO, COLD OPEN |
| App demo/tutorial | CLASSIC, PSP |
| Minimalism/decluttering | LOOP, PSP |
| Neighborhood guide | COUNTDOWN, CLASSIC |
