# Givore Thumbnail Generator (Canva)
## Claude Project Instructions

---

## Task
Create a thumbnail/cover image in Canva for Givore social media videos using the provided video frame and title.

## Input Required
1. **Video frame**: Image file (screenshot from video showing the best moment/item)
2. **Title**: The video title (from any platform - will be adapted for thumbnail)
3. **Platform** (optional): TikTok, Instagram, YouTube, Facebook, LinkedIn (affects size)

---

## Givore Brand Identity

### Brand Elements
- **Name**: Givore
- **Tagline**: "Share, Find, Give"
- **Mascot**: Cat character (multiple versions available)
- **Mission**: Social recycling - give items a second life

### Mascot Characters (Use Sparingly)
Available cat mascots from givore.com:
- `kitty-found.png` - For "found items" content
- `kitty-searching.png` - For "looking for" content
- `kitty-gift.png` - For "give away" content
- `kitty-working.png` - General use
- `kitty-love.png` - Community/emotional content

### Colors
| Use | Color | Hex | Notes |

### Fonts
| Element | Font | Style |
|---------|------|-------|
| Main title | Poppins or Montserrat | Bold / Extra Bold |
| Subtitle | Poppins or Montserrat | Semi Bold |
| Body | Open Sans | Regular |

### Logo Usage
- **Primary**: Givore logo with cat icon
- **Placement**: Bottom-right corner (preferred) or top-left
- **Size**: Max 10-15% of canvas width
- **Versions**:
  - Full color on light backgrounds
  - White on dark backgrounds/photos

---

## Thumbnail Design Rules

### Text Placement
- **Main text**: Lower third OR center of image
- **Never cover**: The main item/subject being shown
- **Safe zones**: Keep text away from edges (10% margin)

### Text Styling
- **Size**: Large enough to read on mobile (minimum 1/4 of frame height)
- **Stroke**: 4-6px black outline on white text
- **Shadow**: Optional drop shadow for depth
- **Maximum words**: 5-7 words (shorter = better)

### Background Treatment
- If frame is busy: Add semi-transparent dark overlay (20-40% black)
- If frame is clear: Use as-is with text overlay
- Ensure contrast between text and background

## Canva Creation Steps

### Step 1: Create Design
- Open Canva
- Select correct size based on platform
- OR use "Givore Thumbnail Template" if available

### Step 2: Set Background
- Upload the provided video frame
- Set as background (fill entire canvas)
- If too busy, add dark overlay:
  - Add rectangle covering full canvas
  - Set color to black (#000000)
  - Set transparency to 20-40%

### Step 3: Add Title Text
- Add text element
- Apply brand font (Montserrat Extra Bold)
- Set color to white (#FFFFFF)
- Add effects:
  - Outline: 4-6px black
  - Shadow: optional, subtle
- Position in lower third or center (not covering main subject)

### Step 4: Add Logo (Optional)
- Add Givore logo
- Position: bottom-right corner
- Size: Small (max 10% of canvas width)
- Opacity: 80-100%

### Step 5: Final Check
- [ ] Text is readable on mobile
- [ ] Main subject visible
- [ ] Text has good contrast
- [ ] Logo present but subtle
- [ ] Correct size for platform
