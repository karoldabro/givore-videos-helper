# Trial CTA Variations

Calls to action specific to trial content. Organized by goal and marketing approach compatibility.

---

## CTA Selection Decision Tree

```
¿Qué marketing approach tiene este trial?

├─ INDIRECT (sin mención de Givore)
│   ├─ Quiero comentarios → ENGAGEMENT
│   ├─ Quiero seguidores → FOLLOW
│   └─ Quiero alcance → SAVE/SHARE
│
├─ SOFT (mención sutil al final)
│   ├─ Quiero que visiten el perfil → PROFILE-VISIT
│   ├─ Quiero que guarden → SAVE
│   └─ Quiero que compartan → SHARE
│
└─ DIRECT (Givore como solución)
    ├─ Quiero descargas → DOWNLOAD
    ├─ Quiero concienciación → AWARENESS
    └─ Quiero que guarden para después → SAVE
```

## CTA Rotation Rules

1. **Never repeat** the same CTA type in 2 consecutive trial scripts
2. **Vary** between tag-based, comment-based, and other subtypes
3. **Match** CTA to the content tone (sarcastic content + empathetic CTA = jarring)
4. **For INDIRECT content:** prioritize ENGAGEMENT and FOLLOW
5. **For SOFT content:** prioritize SAVE and PROFILE-VISIT
6. **For DIRECT content:** prioritize DOWNLOAD and SAVE

---

## Anti-Commercial Rules (Same as Street-Finds)

### NEVER
- "Descargad la app ahora"
- "Link en bio"
- "No os olvidéis de descargar"
- Multiple CTAs pushing the same action
- CTA longer than 2 sentences

### ALWAYS
- CTA matches the content tone
- CTA feels natural, not forced
- Brief -- last 3-5 seconds maximum
- One CTA per video (not three)
